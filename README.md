# 一些常用的 js 方法。

## 立即开始

#### 安装

```bash
npm i pr-tools
```

#### 引入

```js
// 按需引入
import { uuid, md5, regTest, timeFormat, timeFrom, line2hump, hump2line } from './tools/index.js'
// 或全量引入
import * as prTools from './tools/index.js'
```
